# Edge IntelliShield

AI-powered Fraud Detection Web Application using:

- Python
- Flask
- HTML
- CSS
- Scikit-learn
- Random Forest Algorithm

## Features

- Detect fraudulent transactions
- Web-based interface
- Real-time prediction
- Machine Learning integration

## Installation

Install libraries:

pip install -r requirements.txt

## Train Model

python train_model.py

## Run Application

python app.py

## Open Browser

http://127.0.0.1:5000
